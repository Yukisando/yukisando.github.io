My Life

My dream job, for as long as I can remember, has been to work in the video game industry. As an enthusiast gamer and a musician, I oriented my studies towards becoming a sound engineer as well as an avid coder.Currently learning Java, IPython, JavaScript, processing, and more importantly C++, I feel I now have the tools to get closer to my ultimate goal.Most of my free time is spent coding for games, creating plugins for Minecraft, World of Warcraft (Java and LUA) and making VSTs and compositions using MAX MSP, Ableton Live and Pro Tools.I traveled a lot during my youth, having lived in Canada, Singapore, the Netherlands, France, Belgium, Switzerland and more recently the UK. These moves greatly improved my communication skills and reinforced my open-mindedness.

My work

One of the things I particularly enjoy doing in my free time, is to grab a common object (plastic bottle, Glass, Pencil...) and extract as many sounds as I can from it. I then put all of these samples into a sampler and compose.

Portfolio

Shooter Hero

Shooter Hero is a 2D shooter game made entirely on C++ using OF library. Part of a second year project, I had to make a C++ application using inheritance, polymorphism, concurrency and smart pointers. Working on this game allowed me to better organize the hierarchy of my code and make sure it is running as fast and safely as possible.


Github repo: Shooter Hero

Listen

A 3D virtual reality horror game. Left in the dark, hunted by a mysterious ghost-like entity, you must navigate using your echolocation skills to locate the light switch undetected

DS4_2_Synth
